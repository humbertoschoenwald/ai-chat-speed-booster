# Edge install guide

## Store install

There is no separate Microsoft Edge Add-ons listing yet. Edge can install compatible Chrome Web Store extensions when **Allow extensions from other stores** is enabled:

- https://chromewebstore.google.com/detail/ai-chat-speed-booster/fgefgkfmapdjjjdekejanelknedclfik

## Manual developer install

1. Download the latest Edge release asset from:
   - https://github.com/Noah4ever/ai-chat-speed-booster/releases
2. Extract the zip on your computer.
3. Open `edge://extensions`.
4. Enable **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted `dist/edge` folder.
